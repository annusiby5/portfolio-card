# Personal Profile Card

A clean and responsive personal profile card built using **HTML** and **CSS**. This project showcases a developer profile with a modern dark theme, profile image, skills, and social links.

## Preview

A simple portfolio-style card featuring:

* Profile picture
* Name and role
* Availability status
* Short bio
* Tech stack
* Social media links
* Responsive design

## Built With

* HTML5
* CSS3
* Google Fonts (Inter)
* Tabler Icons

## Features

* Modern dark UI
* Responsive layout
* Animated status indicator
* Hover effects on skills and social buttons
* Mobile-friendly design
* Easy to customize

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/profile-card.git
```

2. Open the project folder:

```bash
cd profile-card
```

3. Open `index.html` in your browser.

## Customization

You can easily personalize the card by updating:

* Profile image
* Name and role
* Bio
* Skills
* Social links
* Theme colors
